 🐾 まこペット 開発ノート

## 🎯 コンセプト

**勉強したくなるではなく、猫に会いたくなるアプリ。**

ドリルを進めると、ふしぎなたまごからかわいい子が生まれて、育っていく。  
勉強は「やらされるもの」ではなく、猫にごはんをあげたり、おもちゃや家具を増やしたりするための冒険にする。

---

## ✅ 決定事項

### 1. ペット
- 最初は **ふしぎなたまご**
- 恐竜の卵みたいな見た目
- たまごから、いろんなかわいい猫が生まれる
- 猫はあとから増やせる
- 図鑑に登録される

### 2. 表示メッセージ
ホーム画面に入れたい言葉：

> ふしぎなたまごから かわいい子がうまれるよ

### 3. 勉強ポイント
一科目ごとにポイントをつける。

| 状態 | ポイント |
|---|---:|
| やらなかった | 0 |
| 少しやった | 1 |
| 目標どおりできた | 2 |
| 目標以上できた | 3 |

完璧じゃなくても、少しやったらちゃんと認める。

### 4. ポイントの使い道
掃除はなし。かわいい方向に使う。

- 🍎 ごはん
- 🧸 おもちゃ
- 🛏 ベッド
- 🛋 家具
- 🎀 着せ替え
- 🐟 おやつ
- ✨ レアたまご

### 5. 着せ替え
- リボン
- 帽子
- ワンピース
- メガネ
- 羽
- 季節イベント衣装

### 6. お部屋
- ベッド
- クッション
- キャットタワー
- 窓
- 壁紙
- 床
- おもちゃ
- 観葉植物

## 🧭 大事にする方針

- 子どもを責めない
- やらなかった日があっても猫はいなくならない
- 死なない、病気にならない
- 罰ゲームより、楽しいごほうびを増やす
- 「少しやった」もちゃんと評価する
- 課金しなくても楽しいゲームにする
